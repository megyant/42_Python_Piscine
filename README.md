# Python Piscine

This repository consists of a set of smaller projects to introduce programming to python and object-oriented programming. There are different modules ranging from 0 to 11, each composed of several different exercises, from basic to advanced.

## Module 00

This module contains 7 exercises. Here we learn about how defining functions in python works, about how to work with strings and ints and how to use some functions in python such as print() ant input().

## Module 01

This module contains 6 exercises. Here we learn about how to work with classes and subclasses, allowing for the creation of more complex programs.
